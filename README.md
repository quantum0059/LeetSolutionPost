# LeetCode Solution Post Generator

A Chrome extension that automatically generates high-quality, detailed solution posts for LeetCode problems using AI.

## Features

- **Multi-Language Support**: Generates code implementations in Python, Java, C++, and JavaScript from your solution
- **Professional Formatting**: Follows LeetCode's post format with tabs, inline code, and proper markdown
- **Humanized Explanations**: Conversational, easy-to-understand explanations with examples
- **Complete Posts**: Includes Intuition, Approach, Complexity Analysis, and Code sections
- **Flowcharts**: Visual algorithm flowcharts for better understanding
- **Upvote Appeal**: Ends each post with a polite upvote request to boost engagement

## Installation

1. Clone or download this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" in the top-right corner
4. Click "Load unpacked" and select the extension folder
5. Configure your Google Gemini API key in the extension options

## Usage

1. Solve a LeetCode problem and copy your solution code
2. Click the "Gen Post" floating button on the problem page
3. Paste your solution and add approach names
4. Click "Generate Post"
5. Copy the generated content and paste it into LeetCode's post editor
6. Submit your solution post!

## Requirements

- Google Gemini API Key (free tier available)
- Chrome browser

## How It Works

The extension analyzes your solution code and problem description, then uses AI to generate:
- Detailed intuition and approach explanations
- Step-by-step algorithm breakdown
- Time and space complexity analysis with reasoning
- Complete working code in 4 languages
- Professional formatting matching top LeetCode contributors
