About

LeetCode Solution Post Generator v1.0.0

This extension helps you create detailed, well-structured solution posts for LeetCode problems. Simply solve a problem, select your solution code(s), and let AI generate a comprehensive explanation with time/space complexity analysis.

Your API key is stored locally and never sent anywhere except directly to Google's Gemini API.
